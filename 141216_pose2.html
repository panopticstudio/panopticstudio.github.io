Download
  • Use the getData.sh script in the PanopticStudio Toolbox to download all the available data of the 141216_pose2 sequence

    ./getData.sh 141216_pose2
  • Refer to the Documentation and the PanopticStudio Toolbox on GitHub
  • The following table shows currently available data of 141216_pose2 sequence

    • Calibration Data (VGAs + HDs + Kinects) Available
    VGA (457 videos, compressed using crf 10) Available
    HD (19 videos, compressed using crf 20) Available
    3D Pose COCO19 (compatible to VGA frame index, by per-frame reconstruction) Available
    3D Pose COCO19 (compatible to HD frame index, by per-frame reconstruction) Available
    3D Face (compatible to HD frame index) Not available yet
    3D Hand (compatible to HD frame index) Not available yet
    Kinect Depth Not available yet
    Kinect RGB Not available yet
Note
  • This seuqnece has been captured at the early stage of our system and it has several issues.
    - Some HD cameras are vertical: 00_02, 00_04, 00_13
    - Some HD calibration parameters are wrong and should be ignored: 00_14, 00_20, 00_28, 00_29
    - Some VGA calibration parameters are missing: 04_08, 11_04, 14_18, 16_14, 17_06, 20_21
    - Some HD cameras are missing: 00_06, 00_07, 00_15, 00_26, 00_29, 00_30
