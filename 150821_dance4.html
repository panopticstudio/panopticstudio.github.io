Download
  • Use the getData.sh script in the PanopticStudio Toolbox to download all the available data of the 150821_dance4 sequence

    ./getData.sh 150821_dance4
  • Refer to the Documentation and the PanopticStudio Toolbox on GitHub
  • The following table shows currently available data of 150821_dance4 sequence

    • Calibration Data (VGAs + HDs + Kinects) Available
    VGA (480 videos, compressed using crf 10) Available
    HD (29 videos, compressed using crf 20) Available
    3D Pose (compatible to VGA frame index, by per-frame reconstruction) Not available yet
    3D Pose (compatible to HD frame index) Not available yet
    3D Face (compatible to HD frame index) Not available yet
    3D Hand (compatible to HD frame index) Not available yet
    Kinect Depth Not available yet
    Kinect RGB Not available yet
